Who Am I?

Welcome! My name's Eddie Abbondanzio, and this is my personal blog. I'm a software developer who loves to work on his own projects whenever possible. I graduated from the University of Maine in Orono in 2018 with a BS in Computer Science. I'm currently self-employed as an automotive detailer and run my own business: Ed's Auto Detailing. Blog posts will cover anything from software architecture, design patterns, web development, and occasionally updates from working on my cars.

Site Tech Stack

This site is built using Hugo. While I'm not too familar with the Go language, I'm learning as I go (haha). I'm not the biggest fan of WordPress and prefer to run a static site since this blog doesn't need any stellar features. Most of the site is built with the classic web trio (html, css, js), and posts are written using markdown files. I highly recommend it for anyone looking for a good static site framework.
