Site Tech Stack

This site is built using Hugo. While I’m not too familar with the Go language, I’m learning as I go (haha). I’m not the biggest fan of WordPress and prefer to run a static site since this blog doesn’t need any stellar features. Most of the site is built with the classic web trio (html, css, js), and posts are written using markdown files. I highly recommend it for anyone looking for a good static site framework.
